# AgendaFácil

Sistema de agendamento desenvolvido em PHP, HTML, CSS e JavaScript.

## Funcionalidades

- Cadastro de clientes e serviços
- Controle de data e horário
- Bloqueio de horários duplicados
- Confirmação, cancelamento e exclusão
- Filtro por status
- Proteção CSRF
- Armazenamento local em JSON com bloqueio de escrita
- Layout responsivo

## Requisitos

- PHP 8.0 ou superior

## Como executar

Na pasta do projeto, execute:

```bash
php -S localhost:8000
```

Depois, acesse `http://localhost:8000`.

O arquivo de dados é criado automaticamente na primeira execução.
